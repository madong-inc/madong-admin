数据库信息

请输入数据库服务器地址，一般为：localhost
请输入数据库端口号，一般为：3306
请输入数据库的名称，如果没有请先新增
请输入数据库账号，默认为：root
请输入数据表前缀，默认为：ma_
请输入连接数据库账号的密码
请选择是否强制覆盖数据库

Redis 信息

请输入redis服务器地址，一般为：localhost
请输入redis端口号，一般为：6379
请输入redis密码 未设置密码忽略

管理信息

请输入后台管理入口地址： admin
请输入企业名称
请输入后台管理员的登录账号
请输入后台管理员的登录密码
返 回 开始安装
